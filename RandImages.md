# RandImages Help

Send random images (animals, art ...) from different APIs.

# art
 - Usage: `[p]art `
 - Cooldown: `1 per 0.5 seconds`

Send art from random subreddits.

# birb
 - Usage: `[p]birb `
 - Cooldown: `1 per 0.5 seconds`

Send a random birb image from alexflipnote API.

# cat
 - Usage: `[p]cat `
 - Aliases: `cats`
